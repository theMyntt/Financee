# Financee

Aplicativo de finanças, desenvolvido com React Native, NestJS e MongoDB.

## Pre-requisitos

- [API do Financee](https://github.com/themyntt/financee-api)
- NodeJS >= 21.5.0
- MongoDB
- Yarn

## Inicialização

```bash
# clone repo with
git clone https://github.com/themyntt/financee

# install the dependencies with
yarn install

# run program with
yarn run start
```
